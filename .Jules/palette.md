## 2024-04-28 - Missing Explicit Labels and Role Alert in Authentication Forms\n**Learning:** The authentication forms (login and register) were missing explicit `<label>` tags with `for` attributes connecting to input `id`s, and dynamic form validation messages lacked `role="alert"`. This makes it difficult for screen reader users to understand what input is required and if they have made errors.\n**Action:** Always ensure that input fields have explicitly associated `<label>` tags (using `for`/`id` pairs) and that all dynamic error/validation messages use `role="alert"` so they are reliably announced by assistive technologies.
